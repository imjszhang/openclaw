# OpenClaw 实践日记

> 基于视角：[P23-practice-diary](../../pyramid/structure/P23-practice-diary/)

## 这个系列讲什么

37 天从零开始 fork、改造、并在 OpenClaw 上构建自己工具的真实过程记录。不是教程，而是实践中发生的事：摸底一个陌生的大型项目、深入各子系统、做出两个扩展（js-eyes、js-knowledge-prism）、处理两次大规模上游合并，以及中间踩过的各种坑。

时间跨度：2026-01-31 至 2026-03-08

## 适合谁

- 想深度实践开源 AI 框架（不只是用，而是改造它）的开发者
- 对"如何系统接手并演化一个大型开源项目"感兴趣的人
- OpenClaw 的潜在贡献者或重度用户

## 怎么读

每篇对应一个有记录的实践日期，可以顺序阅读，也可以按感兴趣的日期跳读。顺序阅读能感受到完整的认知演化弧线；跳读时建议先看 [INDEX.md](INDEX.md) 找到感兴趣的主题。
