# 个人知识库的结构化方法

> 基于视角：[P01-knowledge-org-methodology](../../pyramid/structure/P01-knowledge-org-methodology/)

## 这篇文章讲什么

如何把按时间线积累的散乱笔记，转化为一套可快速检索、高效复用、可持续演化的结构化知识体系。

核心方案是三件事：三层分离架构、增长驱动的组织策略、文档化的增量流程。

## 适合谁

- 有持续记笔记习惯，但发现笔记越多越难用的学习者
- 已有按日期/主题记录的笔记习惯，想把知识整理成可复用体系的人
- 不要求了解金字塔原理（文中会介绍）
