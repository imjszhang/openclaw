### Key Line 间的逻辑关系

```
KL01 (认识两只龙虾——展示效果)
  │
  ├── KL02 (怎么开始养)
  │     └── KL03 (接入日常通讯)
  │
  ├── KL04 (自动化觉醒)
  │     └── KL05 (浏览器控制)

  ├── KL06 (创造者跃迁)
  │     └── KL07 (生态成长)
  │
```

- **KL01→KL02→KL03**：从"看到龙虾在干什么"到"我也想养一只"到"让它住进我的微信"。
- **KL04→KL05**：能力递进。从"它会主动干活"到"它能操控你的工具"。
