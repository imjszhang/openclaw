# P24 arch overview

## 读者

架构师、技术经理、研发团队成员 | 核心诉求：如何构建一个高效的、以 AI Agent 为中心的架构

## SCQA 摘要

- **S**: 随着人工智能技术的发展，越来越多的企业开始将 AI 集成到他们的产品和服务中。这种集成不仅提高了效率，还增强了用户体验。然而，传统的架构模式往往不能很好地适应 AI 的需求，导致项目效率低下和资源浪费。
- **C**: 在这种背景下，出现了一个矛盾：尽管 AI 技术提供了巨大的潜力，但缺乏一个有效的架构模式来充分利用这些技术，特别是在需要结构化调用工具和项目能力的场景中。
- **Q**: 如何设计一个以 AI Agent 为中心的架构，以优化工具和项目能力的结构化调用，从而提高整体项目效率和效果？
- **A**: 采用 Agent-First 架构模式，以 AI Agent 为中心，优化工具和项目能力的结构化调用，可以构建一个高效的、适应 AI 需求的系统。
