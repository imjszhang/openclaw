# KL01: Knowledge Prism 是一种将日常笔记转化为结构化知识的高效工作方法。

> 所属视角：P01-arch-overview
> 上层论点：塔尖

## 支撑论点

### 1.1: Knowledge Prism 通过其三层架构，将散乱的笔记转化为清晰、有条理的知识输出，解决了笔记越多越难用的问题。

- 逻辑顺序：结构
- 引用 atoms: KP-01, KP-02, KP-03, KP-04, KP-05
- 引用 groups: G01

Knowledge Prism 的三层架构包括 journal、pyramid 和 outputs，这种结构化方法使得知识管理更加高效。

### 1.2: journal 层按日期归档的学习笔记，保留完整的探索过程。

- 逻辑顺序：时间
- 引用 atoms: KP-06
- 引用 groups: G01

journal 层作为 Knowledge Prism 的输入层，记录了学习笔记和探索过程，为后续的结构化处理提供了基础。

### 1.3: pyramid 层是整个体系的核心引擎，分为自下而上和自上而下两条轨道。

- 逻辑顺序：程度
- 引用 atoms: KP-07
- 引用 groups: G01

pyramid 层作为 Knowledge Prism 的核心，通过两条轨道的处理，将 journal 层的笔记转化为结构化的知识。

### 1.4: outputs 层基于 pyramid 视角产出的成品。

- 逻辑顺序：时间
- 引用 atoms: KP-08
- 引用 groups: G01

outputs 层作为 Knowledge Prism 的输出层，根据 pyramid 层的处理结果，产出最终的知识产品。

## 论点间关系

支撑论点 1.1 描述了 Knowledge Prism 的整体架构和目标，1.2 和 1.3 分别解释了三层架构中的 journal 层和 pyramid 层的功能和作用，1.4 则说明了 outputs 层的产出。这些论点按照结构逻辑顺序排列，共同支撑了 Key Line 论点。
