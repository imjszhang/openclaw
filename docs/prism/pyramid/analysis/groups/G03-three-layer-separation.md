# G03: 知识库应分为输入（journal）、处理（pyramid）、输出（outputs）三层，各层职责单一

> 原始素材、结构化拆解、面向读者的产出分属不同职责层，混合放置会导致边界模糊。教程等产出物是拆解的输出，不应与拆解过程同级。
