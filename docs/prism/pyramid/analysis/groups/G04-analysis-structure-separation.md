# G04: 素材分析（analysis）跨视角共享，表达组织（structure）按视角独立，两者的终点与起点不等价

> 自下而上的 analysis 是"发现了什么"，自上而下的 structure 是"要传达什么"。前者是所有视角共享的素材池，后者是各视角独立的组织方案。按视角分目录而非按产物类型分，因为不同读者看到的金字塔结构可能完全不同。

## 组内逻辑顺序

结构顺序：先建立 analysis 与 structure 的宏观分离（KA-09），再阐明两者交接处的语义差异（KA-10），最后展开 structure 内部的视角分目录策略（KA-12）。
