# G05: 产物是否随素材规模增长决定组织方式——增长型用目录分维度，固定型用单文件保内聚

> 知识库中不同产物的增长特性差异显著：atoms 和 tree 会持续膨胀需要目录化拆分，SCQA 是固定结构的逻辑链不应拆开。以增长预期驱动组织决策可以兼顾扩展性和内聚性。

## 组内逻辑顺序

程度顺序：先给出通用判断准则（KA-14），再用具体产物示例印证——增长型的 atoms（KA-11）和 tree（KA-13）做目录拆分，固定型的 SCQA（KA-15）保持单文件。排列时先决策原则后具体案例。
