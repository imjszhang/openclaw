## 与 structure/ 的衔接

synthesis 产出的是"从素材中发现了什么"（分析视角）。进入 structure/ 后，需要从"面对读者要传达什么"（表达视角）重新审视这些候选：

- 哪些候选直接成为 Key Line？
- 哪些需要合并或降级？
- 是否有读者会追问但 synthesis 中缺失的观点？

差异记录见 [structure/validation.md](../structure/validation.md)。
