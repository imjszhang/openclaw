# G11: 基础设施的安全默认值：镜像固定摘要 + 非 root 运行 + 自动化迁移工具

> 安全不只在应用层，基础设施同样需要"默认安全"：Docker 镜像固定到不可变摘要防止 tag 漂移，构建/测试以最小权限用户运行，配置升级通过自动化迁移工具降低人为出错风险。

## 包含的 Atoms

| 编号  | 来源                       | 内容摘要                                                               |
| ----- | -------------------------- | ---------------------------------------------------------------------- |
| MU-15 | merge-main-upgrade-summary | 旧配置通过 `openclaw doctor --fix` 自动迁移，降低 breaking change 影响 |
| MU-16 | merge-main-upgrade-summary | Docker 镜像固定 SHA256、node 用户构建、E2E 非 root 运行                |

## 组内逻辑顺序

结构顺序：构建时安全（MU-16，镜像固定 + 非 root）→ 升级时安全（MU-15，自动迁移工具）。从静态部署环境到动态版本升级。
