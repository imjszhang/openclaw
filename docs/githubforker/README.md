# OpenClaw 个人实践知识库

个人在学习和使用 OpenClaw 过程中的完整知识体系，包含原始记录、结构化拆解和面向读者的产出三个层次。

```
journal/  （原始素材）
    ↓ 自下而上：打散 → 归组 → 收敛
pyramid/  （结构化拆解）
    ↓ 自上而下：序言 → 展开 → 验证（按视角独立）
outputs/  （面向读者的产出）
```

## 金字塔拆解（pyramid/）

基于《金字塔原理》的两阶段结构化拆解：

- **analysis/**（自下而上，共享）：atoms → groups → synthesis
- **structure/**（自上而下，按视角独立）：每个视角一套 SCQA + tree + validation

→ [拆解工作区详情](pyramid/README.md)
